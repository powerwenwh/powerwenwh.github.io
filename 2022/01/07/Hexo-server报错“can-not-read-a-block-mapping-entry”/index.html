Hexo server报错“can not read a block mapping entry”

发表于 2022-01-07

最近重装系统后使用hexo突然报错can not read a block mapping entry

在网上查了一下主要有两个原因

  1. 配置文件yaml语法不正确，没有在配置项后面加空格
  2. 配置文件中出现了""，导致文章无法按照正确yaml语法解析出错，把英文双引号改成中文双引号即可。
