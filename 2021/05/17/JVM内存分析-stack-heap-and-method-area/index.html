    JVM内存分析-stack, heap and method area
    发布于：2021-05-17 16:36:57 标签：/ Java / 访问：

    JVM的内存分为三个区域：

    • 栈 stack
    • 堆 heap
    • 方法区 method area

    栈 stack（连续的内存空间）

    • 方法执行的内存模型。是一个连续的内存空间。
    • JVM为每个线程创建一个栈。栈属于线程私有，不能共享。？？？线程如何共享信息？？？
    • 每个方法调用都会创建一个栈帧
    • 特点： 先进后出，后进先出

    堆 heap（不连续的内存空间）

    • 存储对象
    • JVM只有一个堆，被所有线程共享！！！线程共享信息的方式！！！
    • 不连续的内存空间。

    方法区 method area

    • 堆的一部分
    • 存储类信息中不变或者唯一的内容
      • 代码
      • 静态变量
      • 静态方法
      • String常量
