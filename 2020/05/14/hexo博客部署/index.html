hexo博客部署
发布于：2020-05-14 20:23:32 标签：/ 写博客 / 访问：

前期准备

  1. Node.js
  2. git

一，安装cnpm

即淘宝镜像源，cnpm国内访问速度会快一点。

npm install -g cpm --registry=https://registry.npm.taobao.org

二，安装hexo

cnpm install -g hexo-cli

三，创建博客

新建空文件夹，在空文件夹下操作。

hexo init

四，启动或预览

hexo s

五，新建文章

hexo n "标题"

六，部署到GitHub

  1. 创建新的仓库，注意名字要和自己GitHub名字一样

  2. 安装git部署插件cnpm install --save hexo-deployer-git

  3. 设置_config.yml文件

  4. 部署命令hexo d（注意部署前git设置用户名和邮箱）
